offsetLeft/offsetTop/offsetHeight/offsetWidth offsetLeft和offsetTop总是找到离其最近的已经定位的元素定位，如果没有，根据body定位，然后获取其left,top值,如果元素的display为none，或者元素的position为fixed,元素的offsetParent为null offsetLeft和offsetTop获取匹配元素在当前视口的相对偏移

offsetParent 返回一个元素离其最近的已经定位的元素

window.outerWidth/window.outerHeight 浏览器窗口的大小，包括顶部菜单栏、滚动条；

window.innerWidth/window.innerHeight 浏览器窗口的大小，不包括顶部菜单栏、滚动条；

document.documentElement.clientwidth 返回整个文档的可见宽度（不包含边框），clientwidth = offsetWidth - borderWidth; 如果不给document.documentElement来设置边框，所以这里的clientwidth 与 offsetWidth一致

document.body.scrollWidth/document.body.scrollHeight 如果文档宽高小于浏览器宽高，scrollWidth/scrollHeight取浏览器可见区域宽高； 如果文档宽高大于浏览器宽高，scrollWidth/scrollHeight取文档的宽度/高度；

clientWidth = width + padding
offsetWidth = width + padding + border
scrollWidth = width + padding (是对象的实际内容的宽，不包边线宽度,不随窗口大小改变；但在chrome下包含边框；) 如果没有边线，两者相等；
